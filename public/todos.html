✅ Setup Checklist

Complete these steps to secure your Supabase database

Add SUPABASE_SERVICE_KEY on Vercel

Go to Vercel Dashboard → Project → Settings → Environment Variables and add:

SUPABASE_SERVICE_KEY = your Supabase service_role key

Get the key from Supabase Dashboard → Project Settings → API → service_role key (not the anon key).
