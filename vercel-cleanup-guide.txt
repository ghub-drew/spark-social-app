VERCEL CLEANUP GUIDE - Spark Social App
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install Vercel CLI (if not already installed)

   Open Command Prompt and run:
   npm install -g vercel

   Verify:
   vercel --version

2. Check Current Linkage

   cd C:\Users\andre\Documents\Deepseek\Code App
   vercel list

   Also check for local .vercel folder:
   dir .vercel

3. Delete Duplicate Projects (via Dashboard - safest)

   Go to https://vercel.com/dashboard
   Find each duplicate and click ... then Delete:
   - cwd-spark-social-app
   - cwd-social-app
   - spark-social-app

4. Remove Local .vercel Link (if exists)

   rmdir /s .vercel

5. Verify Correct Project on GitHub

   Vercel Dashboard -> spark-social-app-ys49 -> Settings -> Git
   Should show: Repo = ghub-drew/spark-social-app, Branch = main

6. Prevent Future Duplicates

   - Never run "vercel link" or "vercel deploy" from project folder
   - Just push to GitHub -> Vercel auto-deploys
   - If a duplicate appears, delete from dashboard

Quick Reference:
   Install CLI:    npm install -g vercel
   List projects:  vercel list
   Check link:     dir .vercel
   Remove link:    rmdir /s .vercel
   Delete project: Dashboard -> ... -> Delete
   Live URL:       https://cebuspark.com
